# Frontend

This folder contains frontend applications for the Library Management project.

## Prerequisites

Run from the project root (`LibraryManagement`):

```powershell
python -m pip install -r requirements.txt
```

## Launch Pattern (Primary)

```powershell
streamlit run frontend/streamlit_app/app.py
```

## Launch Pattern (Module Wrapper)

```powershell
python -m frontend.streamlit_app
```
